## 🔗 معلومات المستودع

## 📝 الأوامر الأساسية (3 فقط!)

### 1️⃣ المزامنة السريعة
```bash
pnpm sync
```
**استخدمه:** بعد كل تعديل على الكود

---

### 2️⃣ النسخ الاحتياطي
```bash
pnpm backup
```
**استخدمه:** قبل التعديلات الكبيرة

---

### 3️⃣ النسخ الاحتياطي + المزامنة معاً
```bash
pnpm backup:auto
```
**استخدمه:** في نهاية يوم العمل

---

## 🎯 الاستخدام اليومي

### صباحاً (قبل البدء)
```bash
cd /home/ubuntu/joniah_pharmacy
pnpm sync
```

### مساءً (بعد الانتهاء)
```bash
cd /home/ubuntu/joniah_pharmacy
pnpm backup:auto
```
