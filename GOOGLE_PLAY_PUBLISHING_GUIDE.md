# دليل نشر تطبيق جونيا للتوصيل على Google Play Store

## نظرة عامة

هذا الدليل يشرح خطوات نشر تطبيق **جونيا للتوصيل** على متجر Google Play. يتطلب النشر حساب مطور Google Play نشط (رسوم التسجيل 25 دولار أمريكي لمرة واحدة).

---

## المتطلبات الأساسية

| المتطلب | الوصف |
|---------|-------|
| حساب مطور Google Play | حساب نشط على [Google Play Console](https://play.google.com/console) |
| ملف AAB أو APK موقّع | ملف التطبيق موقّع بمفتاح إصدار (Release Key) |
| أيقونة التطبيق | صورة 512×512 بكسل بصيغة PNG |
| لقطات شاشة | على الأقل 2 لقطات شاشة للهاتف (1080×1920 بكسل) |
| وصف التطبيق | وصف قصير (80 حرف) ووصف كامل (4000 حرف) |
| سياسة الخصوصية | رابط URL لصفحة سياسة الخصوصية |

---

## الخطوة 1: إنشاء مفتاح التوقيع (Keystore)

لنشر التطبيق على Google Play، يجب توقيعه بمفتاح إصدار. قم بتنفيذ الأمر التالي على جهازك:

```bash
keytool -genkey -v -keystore joniah-release-key.jks -keyalg RSA -keysize 2048 -validity 10000 -alias joniah
```

> **تحذير هام**: احتفظ بملف `joniah-release-key.jks` وكلمة المرور في مكان آمن. فقدان هذا الملف يعني عدم القدرة على تحديث التطبيق مستقبلاً.

عند تنفيذ الأمر، سيُطلب منك إدخال المعلومات التالية:

| الحقل | القيمة المقترحة |
|-------|-----------------|
| كلمة مرور Keystore | اختر كلمة مرور قوية |
| الاسم الأول والأخير | اسمك الكامل |
| اسم الوحدة التنظيمية | Joniah Pharmacy |
| اسم المنظمة | Joniah Pharmacy |
| المدينة | مدينتك |
| المحافظة | محافظتك |
| رمز الدولة | YE (لليمن) |

## الخطوة 2: بناء ملف AAB للإصدار

Google Play يفضل صيغة **Android App Bundle (AAB)** على APK. لبناء ملف AAB:

### 2.1 تحديث ملف build.gradle

أضف إعدادات التوقيع في ملف `android/app/build.gradle`:

```gradle
android {
    ...
    signingConfigs {
        release {
            storeFile file('path/to/joniah-release-key.jks')
            storePassword 'YOUR_KEYSTORE_PASSWORD'
            keyAlias 'joniah'
            keyPassword 'YOUR_KEY_PASSWORD'
        }
    }
    
    buildTypes {
        release {
            signingConfig signingConfigs.release
            minifyEnabled true
            proguardFiles getDefaultProguardFile('proguard-android-optimize.txt'), 'proguard-rules.pro'
        }
    }
}
```

### 2.2 بناء ملف AAB

```bash
cd android
./gradlew bundleRelease
```

سيتم إنشاء الملف في: `android/app/build/outputs/bundle/release/app-release.aab`

## الخطوة 3: إعداد صفحة التطبيق على Google Play Console

### 3.1 إنشاء تطبيق جديد

1. سجّل الدخول إلى [Google Play Console](https://play.google.com/console)
2. اضغط على **"إنشاء تطبيق"** (Create app)
3. أدخل المعلومات التالية:

| الحقل | القيمة |
|-------|--------|
| اسم التطبيق | جونيا للتوصيل |
| اللغة الافتراضية | العربية |
| نوع التطبيق | تطبيق |
| مجاني/مدفوع | مجاني |

4. وافق على الإعلانات والشروط

### 3.2 إعداد قائمة المتجر (Store Listing)

انتقل إلى **"تواجد المتجر"** > **"القائمة الرئيسية للمتجر"**:

#### معلومات التطبيق

| الحقل | المحتوى المقترح |
|-------|-----------------|
| **الوصف القصير** | تطبيق توصيل طلبات صيدلية جونيا - اطلب أدويتك بسهولة |
| **الوصف الكامل** | تطبيق جونيا للتوصيل هو الحل الأمثل لطلب الأدوية والمستلزمات الطبية من صيدلية جونيا. يتيح لك التطبيق تصفح المنتجات، إضافتها للسلة، وطلب التوصيل لموقعك مباشرة. مميزات التطبيق: طلب سهل وسريع، تتبع حالة الطلب، إشعارات فورية، دعم العملاء. |

#### الأصول المرئية

| النوع | المواصفات |
|-------|-----------|
| أيقونة التطبيق | 512×512 بكسل، PNG، بدون شفافية |
| صورة مميزة | 1024×500 بكسل (اختياري) |
| لقطات شاشة للهاتف | 2-8 صور، 1080×1920 بكسل |
| لقطات شاشة للتابلت | 1-8 صور (اختياري) |

### 3.3 تصنيف المحتوى

انتقل إلى **"السياسة"** > **"تصنيف المحتوى"**:

1. ابدأ الاستبيان
2. أجب على الأسئلة بصدق (التطبيق لا يحتوي على محتوى عنيف أو للبالغين)
3. احفظ التصنيف

### 3.4 إعدادات التطبيق

انتقل إلى **"السياسة"** > **"إعدادات التطبيق"**:

| الإعداد | القيمة |
|---------|--------|
| الفئة | صحة ولياقة بدنية > طبي |
| البريد الإلكتروني للتواصل | بريدك الإلكتروني |
| سياسة الخصوصية | https://joniah.xyz/privacy |

## الخطوة 4: رفع التطبيق

### 4.1 إنشاء إصدار

1. انتقل إلى **"الإصدار"** > **"الإنتاج"**
2. اضغط على **"إنشاء إصدار جديد"**
3. ارفع ملف AAB
4. أدخل ملاحظات الإصدار:

```
الإصدار الأول من تطبيق جونيا للتوصيل:
- طلب الأدوية والمستلزمات الطبية
- تتبع حالة الطلبات
- إشعارات فورية
- تحديد موقع التوصيل
```

### 4.2 مراجعة وإطلاق

1. راجع جميع الأقسام وتأكد من اكتمالها (علامة خضراء)
2. اضغط على **"بدء الطرح للإنتاج"**
3. انتظر مراجعة Google (عادة 1-7 أيام)

---

## الخطوة 5: متطلبات إضافية

### 5.1 سياسة الخصوصية

يجب إنشاء صفحة سياسة خصوصية على موقعك. يمكنك إضافتها على:
`https://joniah.xyz/privacy`

يجب أن تتضمن:
- ما البيانات التي يجمعها التطبيق
- كيف تُستخدم البيانات
- مع من تُشارك البيانات
- كيف يمكن للمستخدم حذف بياناته

### 5.2 إعلان أمان البيانات

في Google Play Console، انتقل إلى **"السياسة"** > **"أمان البيانات"**:

| السؤال | الإجابة المقترحة |
|--------|------------------|
| هل يجمع التطبيق بيانات؟ | نعم |
| أنواع البيانات | الموقع، معلومات الاتصال |
| هل تُشارك البيانات مع جهات خارجية؟ | لا |
| هل يمكن للمستخدم طلب حذف بياناته؟ | نعم |

## نصائح مهمة

1. **احتفظ بنسخة احتياطية من Keystore**: فقدانه يعني عدم القدرة على تحديث التطبيق
2. **استخدم Google Play App Signing**: يوفر حماية إضافية لمفتاح التوقيع
3. **اختبر على أجهزة متعددة**: قبل النشر، اختبر التطبيق على أجهزة مختلفة
4. **راقب التقييمات**: رد على تعليقات المستخدمين لتحسين التقييم
5. **حدّث بانتظام**: التطبيقات النشطة تحصل على ترتيب أفضل
