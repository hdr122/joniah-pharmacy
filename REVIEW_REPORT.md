## 🔐 نظام Super Admin

### الميزات المتوفرة

#### 1. لوحة التحكم الرئيسية (`/admin/super-admin`)
- عرض جميع الفروع مع إحصائياتها
- إنشاء فروع جديدة
- تعديل معلومات الفروع
- تجديد اشتراكات الفروع
- عرض الإحصائيات المالية بالدينار العراقي (IQD)

#### 2. نظام كودات الاشتراك (`/admin/subscription-codes`)
- توليد كودات اشتراك جديدة (يومي، أسبوعي، شهري، سنوي، مخصص)
- عرض جميع الكودات المولدة وحالتها
- حذف الكودات
- نسخ الكودات بسهولة
- تحديث سعر الاشتراك

#### 3. الصلاحيات والحماية
- حماية جميع إجراءات Super Admin بـ `superAdminProcedure`
- التحقق من اسم المستخدم `HarthHDR1` أو دور `super_admin`
- عزل البيانات بين الفروع بشكل كامل
- إخفاء قسم إدارة الفروع عن جميع المستخدمين ما عدا Super Admin

## 🔧 الميزات التقنية

### Backend (tRPC + Express)

#### الإجراءات المتوفرة

**Super Admin Routes:**
```typescript
branches.list()           // عرض جميع الفروع
branches.create()         // إنشاء فرع جديد
branches.update()         // تحديث معلومات فرع
branches.getStats()       // إحصائيات جميع الفروع

subscriptions.generateCode()  // توليد كود اشتراك
subscriptions.list()           // عرض جميع الكودات
subscriptions.validateAndUse() // التحقق من كود واستخدامه
subscriptions.delete()         // حذف كود
subscriptions.getPrice()       // الحصول على سعر الاشتراك
subscriptions.updatePrice()    // تحديث سعر الاشتراك
```

**Admin Routes:**
```typescript
deliveryMen.*    // إدارة المندوبين
orders.*         // إدارة الطلبات
customers.*      // إدارة العملاء
statistics.*     // الإحصائيات
users.*          // إدارة المستخدمين
```

### Frontend (React 19 + Tailwind 4)

#### الصفحات الرئيسية

1. **صفحات Super Admin:**
   - `/admin/super-admin` - لوحة التحكم الرئيسية
   - `/admin/subscription-codes` - إدارة كودات الاشتراك

2. **صفحات Admin:**
   - `/admin` - لوحة التحكم
   - `/admin/deliveries` - إدارة المندوبين
   - `/admin/orders` - إدارة الطلبات
   - `/admin/customers` - إدارة العملاء
   - `/admin/statistics` - الإحصائيات المتقدمة
   - وغيرها...

3. **صفحات المندوب:**
   - `/delivery` - لوحة تحكم المندوب
   - `/delivery/notifications` - الإشعارات
   - `/delivery/profile` - الملف الشخصي

## 🔒 نظام المصادقة

### أنواع المستخدمين

1. **Super Admin** (`super_admin`)
   - الوصول الكامل لجميع الفروع
   - إدارة كودات الاشتراك
   - إنشاء وتعديل الفروع
   - المستخدم: `HarthHDR1`

2. **Admin** (`admin`)
   - إدارة فرع واحد فقط
   - الوصول لجميع ميزات الإدارة
   - لا يمكنه رؤية فروع أخرى

3. **Delivery** (`delivery`)
   - الوصول للوحة المندوب فقط
   - رؤية الطلبات المخصصة له
   - تحديث حالة الطلبات

### تسجيل الدخول

- تسجيل دخول مخصص باستخدام اسم المستخدم وكلمة المرور
- دعم OAuth عبر Manus
- جلسات آمنة مع JWT
- مدة الجلسة: 30 يوم
