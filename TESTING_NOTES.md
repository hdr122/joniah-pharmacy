# ملاحظات الاختبار - خريطة تتبع المندوبين المتقدمة

## التاريخ: 2024-12-08

### التحديثات المنجزة:

#### 1. إخفاء الصفحات القديمة ✅
- تم إخفاء صفحة "إعدادات Traccar" من القائمة
- تم إخفاء صفحة "خريطة التتبع" القديمة من القائمة
- تم تغيير اسم "لوحة تحكم Traccar" إلى "خريطة تتبع المندوبين"

#### 2. الخصائص المتقدمة المضافة ✅

**أ. فلتر إخفاء/إظهار المندوبين غير المتصلين:**
- زر "إخفاء غير المتصلين" / "إظهار الكل"
- يعمل بشكل ديناميكي على الخريطة وقائمة المندوبين

**ب. عرض مسار تاريخي للمندوب:**
- قائمة منسدلة لاختيار الفترة الزمنية:
  - الموقع الحالي
  - آخر ساعة
  - آخر 3 ساعات
  - آخر 6 ساعات
  - آخر 12 ساعة
  - آخر 24 ساعة
- رسم المسار على الخريطة باستخدام Polyline
- تكبير تلقائي لعرض المسار بالكامل

**ج. تتبع شامل للطلبات:**
- عرض الطلبات النشطة على الخريطة
- علامات ملونة حسب حالة الطلب:
  - برتقالي: قيد التوصيل (pending)
  - أزرق: في الانتظار
- نافذة معلومات لكل طلب تحتوي على:
  - رقم الطلب
  - اسم المندوب
  - المنطقة
  - السعر
  - الحالة

**د. ضبط عرض الوقت:**
- استخدام توقيت بغداد (Asia/Baghdad) في جميع أنحاء الصفحة
- تنسيق عربي للتاريخ والوقت
- عرض الوقت بصيغة 24 ساعة

**هـ. زر تحديث يدوي:**
- زر "تحديث" مع أيقونة دوارة أثناء التحميل
- رسالة نجاح عند التحديث

**و. عداد للمندوبين:**
- بطاقات إحصائية تعرض:
  - إجمالي المندوبين
  - عدد المتصلين (أخضر)
  - عدد غير المتصلين (أحمر)
  - نسبة الاتصال (%)
  - عدد الطلبات النشطة (أزرق)

#### 3. APIs المضافة ✅

**أ. getDeliveryRoute:**
```typescript
getDeliveryRoute: adminProcedure
  .input(z.object({
    userId: z.number(),
    date: z.string(),
    hours: z.number().optional().default(0),
  }))
  .query(async ({ input }) => {
    const route = await db.getDeliveryRouteByTime(input.userId, input.date, input.hours);
    return route;
  })
```

**ب. getDeliveryRouteByTime في db.ts:**
- يجلب المسار التاريخي للمندوب خلال فترة زمنية محددة
- يدعم الموقع الحالي (hours = 0) أو المسار التاريخي

**ج. getActiveOrdersWithLocations:**
- موجود مسبقاً ويعمل بشكل صحيح
- يجلب الطلبات النشطة مع مواقع المندوبين
