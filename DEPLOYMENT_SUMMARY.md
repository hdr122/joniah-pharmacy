## 📋 Environment Variables Needed

Before deploying, prepare these:

```bash
# Auto-generated by platform
DATABASE_URL=postgresql://...

# Generate new (32+ random characters)
JWT_SECRET=___GENERATE_NEW___

# From OAuth provider
OAUTH_SERVER_URL=https://oauth.manus.im
VITE_APP_ID=___ASK_OAUTH_PROVIDER___
OWNER_OPEN_ID=___ASK_OAUTH_PROVIDER___

# From Google Cloud Console
VITE_GOOGLE_MAPS_API_KEY=___GET_FROM_GOOGLE___

# From Firebase Console (GET NEW KEY!)
FIREBASE_SERVICE_ACCOUNT='{"type":"service_account",...}'

# Fixed values
PORT=3000
NODE_ENV=production
```

**⚠️ IMPORTANT:**
- Generate NEW `JWT_SECRET` (not from .env)
- Get NEW Firebase key (old one in .env is exposed)
- Get Google Maps API key for production
